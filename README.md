# Schürfrecht

![Neale's Stopes, Burra Burra Mine, April 12th 1847 — S. T. Gill](https://upload.wikimedia.org/wikipedia/commons/9/9d/S._T._Gill_-_Neales%27s_Stopes%2C_Burra_Burra_Mine%2C_April_12th_1847_-_Google_Art_Project.jpg)

A browser-based isometric underground mining game. Operate heavy tracked machinery to carve into voxel terrain, clear debris, and push deeper into the earth.

## Run

```bash
npm install
npm run dev
```

## Controls

| Input | Action |
|---|---|
| Left click | Mine (carve terrain) |
| Right drag / Ctrl+drag | Orbit camera |
| Q / E | Snap rotate 90° |
| Scroll | Zoom |
| Ctrl+Scroll | Adjust cutaway depth |
| WASD | Pan |
| Middle drag | Pan |

## Stack

Vite, vanilla JS, Three.js. No external assets — everything is procedurally generated in code.
